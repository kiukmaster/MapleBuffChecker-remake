# 메이플 버프 체커 — Remake

메이플스토리 설치기·버프 스킬의 재사용 타이밍을, 화면에서 아이콘을 실시간으로 감지해
소리로 알려주는 비공식 도우미입니다. 기존 버전을 처음부터 새로 작성한 리메이크입니다.

## 무엇이 달라졌나

- **감지 엔진을 OpenCV.js → WebGL(GPU)로 교체.** 기존에는 ~11MB의 OpenCV WASM을
  내려받아 매 프레임 힙을 갈아치웠고, 이것이 높은 메모리 사용과 알트탭 시 탐색기
  멈춤의 원인이었습니다. 이제 슬라이딩 윈도우 매칭을 프래그먼트 셰이더에서 수행하여
  프레임당 JS 할당이 거의 없고, 캡처 프레임을 `<video>`에서 GPU로 바로 업로드합니다.
- **OpenCV `TM_CCOEFF_NORMED`를 셰이더로 동일하게 재현**(채널별 평균·알파 포함)했기
  때문에 기존에 스킬별로 맞춰둔 정확도 임계값을 그대로 사용할 수 있습니다.
- 코드를 역할별로 분리해 새로 작성했습니다(엔진 / 알림 상태머신 / 오케스트레이션 훅 / UI).

## 구조

```
app/
  layout.js              레이아웃 · 내비게이션 · 푸터
  page.js                랜딩
  detect/page.js         감지 UI (상태만 렌더, 로직은 훅에 위임)
  components/Navbar.js
lib/detection/
  catalog.js             스킬 카탈로그(선언적 도메인 데이터)
  WebGLTemplateMatcher.js GPU 템플릿 매칭 엔진
  AlertController.js     instant / 쿨다운 / 지연 / 타이머 알림 상태머신
  useDetector.js         화면공유 + 감지 루프 + 알림을 묶는 React 훅
public/
  detect/                스킬 템플릿 이미지(해상도·시간별)
  sound/                 알림음
```

## 감지 동작 (특수 케이스 포함)

- **기본(instant)**: 일치율이 임계값을 처음 넘는 순간 1회 알림, 임계값 아래로 내려가면 재무장.
- **쿨다운**: 파운틴 · 야누스는 3초 내 재알림 방지.
- **지연(delayed)**: 마력[레테]는 최초 감지 후 5초 뒤 1회 알림. 대기 중 잠깐 끊겨도 타이머 유지.
- **타이머**: 일격필살은 감지 시 30초 카운트다운을 시작하고 종료 N초 전(5/10초)에 알림,
  10초간 재감지 무시.
- **남은 시간 선택**: 파운틴 · 야누스 · 골누스는 남은 시간별 템플릿으로 감지.
- **동시 선택 금지**: 알레리아 ↔ 헤도.

## 개발

```bash
npm install
npm run dev      # http://localhost:3000
```

> 화면 공유(getDisplayMedia)는 보안 컨텍스트에서만 동작합니다. `localhost` 또는 https로
> 접속하세요. WebGL2 + 부동소수 텍스처가 필요하며, 최신 크롬 / 엣지를 권장합니다.
